## Описание программы

"Book Tracker" - это настольное приложение с графическим интерфейсом, созданное на Python с использованием `tkinter`, предназначенное для ведения учета прочитанных книг. Оно позволяет пользователям добавлять информацию о книгах (название, автор, жанр, количество страниц), просматривать их в виде таблицы, а также фильтровать список по жанру и количеству страниц. Все данные о книгах сохраняются и загружаются из файла `books.json`, обеспечивая постоянство информации. Проект включает настройку локального Git-репозитория для управления версиями.

## Инструкция по запуску

1.  **Установка Python:** Убедитесь, что на вашем компьютере установлен Python (версия 3.6 или выше).
2.  **Сохранение кода:** Скопируйте весь предоставленный код и сохраните его в файле с расширением `.py` (например, `book_tracker.py`).
3.  **Запуск:** Откройте терминал или командную строку, перейдите в директорию, где вы сохранили файл, и выполните команду:
    ```bash
    python book_tracker.py
    ```
4.  **Git:** При первом запуске в директории будет автоматически создан `.git` репозиторий, и настроен файл `.gitignore` для исключения `books.json` и других временных файлов.
5.  **GitHub:** Для выгрузки проекта на GitHub:
    *   Создайте новый репозиторий на GitHub.
    *   В терминале, в директории с вашим проектом, выполните команды:
        ```bash
        git add .
        git commit -m "Initial commit"
        git branch -M main
        git remote add origin <URL вашего репозитория на GitHub>
        git push -u origin main
        ```

## Примеры использования

*   **Добавление книги:**
    *   Введите "Название", "Автор", "Жанр" и "Количество страниц" (число).
    *   Нажмите кнопку "Добавить книгу". Книга появится в таблице.
    *   *Корректность ввода:* Программа выдаст предупреждение, если поля пусты или количество страниц не является числом.
*   **Фильтрация:**
    *   В секции "Фильтр" в поле "Жанр" введите название жанра (например, "Фантастика"). Список книг обновится, показывая только книги с этим жанром.
    *   В поле "Страниц >" введите число (например, 200). Список отфильтруется, показывая книги, у которых страниц больше указанного значения.
    *   Комбинируйте фильтры.
    *   Нажмите "Сбросить фильтр", чтобы вернуть отображение всех книг.
*   **Сохранение данных:** Каждое добавление книги автоматически сохраняется в файл `books.json`. При следующем запуске приложения данные будут загружены из этого файла.
